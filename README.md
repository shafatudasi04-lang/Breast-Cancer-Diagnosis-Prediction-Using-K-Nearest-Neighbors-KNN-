# Breast Cancer Diagnosis Prediction Using K-Nearest Neighbors (KNN)

## Author
**Shafaat Ullah**

## Project Overview
This project implements the K-Nearest Neighbors (KNN) classification algorithm to predict whether a breast tumor is **Benign (B)** or **Malignant (M)** using the Breast Cancer Wisconsin Diagnostic Dataset.

The project includes complete data preprocessing, exploratory data analysis (EDA), feature scaling, model training, evaluation, and prediction.

---

## Dataset

- Dataset: Breast Cancer Wisconsin Diagnostic Dataset
- Total Records: 569
- Features: 30
- Target Variable: Diagnosis
- Classes:
  - Benign (B)
  - Malignant (M)
